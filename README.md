# Timer-Arduino

## Italiano
Un semplice timer.

Realizzazione di un timer con una scheda Arduino UNO R3.
È possibile impostare un timer per poi accenderlo. È inoltre possibile metterlo in pausa, alzare/abbassare il volume e interrompere il timer una volta scattato.
Il tempo verrà mostrato sul display BCD a 4 cifre.

### Componenti necessari
- UNO R3 Controller Board
- 4 Digit 7-Segment Display
- IR Receiver Module
- Remote Control
- Passive Buzzer
- 74HC595 IC
- 5 resistori 220 Ohm
- Breadboard
- Breadboard jumpers

## English
A simple timer.

Implementation of a timer using an Arduino UNO R3 board.
It is possible to set a timer and then start it. You can also pause it, increase/decrease the volume, and stop the timer once it has finished.
The time will be displayed on a 4-digit BCD display.

### Required components
- UNO R3 Controller Board
- 4 Digit 7-Segment Display
- IR Receiver Module
- Remote Control
- Passive Buzzer
- 74HC595 IC
- 5 resistors 220 Ohm
- Breadboard
- Breadboard jumpers
