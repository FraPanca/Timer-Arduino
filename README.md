# Timer-Arduino
Un semplice timer.

Realizzazione di un timer con una scheda Arduino UNO R3.

Componenti necessari:
  - UNO R3 Controller Board
  - 4 Digit 7-Segment Display
  - IR Receiver Module
  - Remote Control
  - Passive Buzzer
  - 74HC595 IC
  - 5 resistor 220 Ohm
  - Breadboard
  - Breadboard jumpers
